# LAVAI — Roadmap Oficial

> Estratégia: vender antes de construir tudo. MVP mínimo que resolve a dor principal.

---

## 🔴 FASE 0 — Validação e Venda (Semana 1–2)

**Objetivo:** Fechar os primeiros 3 clientes ANTES do MVP estar completo.

### Ações imediatas:
- [x] Definir nome e posicionamento → **LAVAI**
- [ ] Criar landing page de pré-cadastro (1 dia)
- [ ] Criar apresentação/pitch para visita presencial (1 dia)
- [ ] Visitar 10 lava-jatos da região
- [ ] Fechar 3 clientes com desconto de early adopter (R$49/mês ou gratuito no 1º mês)
- [ ] Coletar feedback real dos donos sobre as maiores dores

### Entregável:
> Uma landing page rodando + 3 contratos assinados (mesmo que informais)

---

## 🟡 FASE 1 — MVP (Semana 2–4)

**Objetivo:** Sistema funcionando com o mínimo que resolve a dor principal.

### Features obrigatórias do MVP:
- [ ] **Dashboard do lava-jato** (web, mobile-first)
  - Fila de clientes em tempo real
  - Status de cada serviço (aguardando / em andamento / finalizado)
  - Cadastro rápido de cliente (nome + placa + serviço)
- [ ] **Agendamento básico**
  - Link público para o cliente agendar
  - Confirmação via WhatsApp (Evolution API)
- [ ] **Cadastro de clientes** com histórico de visitas
- [ ] **Relatório simples** — serviços do dia, receita total

### Features fora do MVP (não construir agora):
- ~~App mobile nativo~~
- ~~Web3 / tokens~~
- ~~IA preditiva~~
- ~~Multi-unidades~~
- ~~Pagamento integrado~~

---

## 🟢 FASE 2 — Produto (Mês 2–3)

**Objetivo:** Transformar o MVP em produto real com 20 clientes ativos.

- [ ] **IA de agendamento inteligente** — sugere horários baseado em histórico
- [ ] **WhatsApp Bot** completo (agendar, consultar fila, notificar quando pronto)
- [ ] **Programa de fidelidade** — pontos por visita, resgate de desconto
- [ ] **Relatórios de desempenho** — comparativo semanas/meses
- [ ] **Sistema de pagamento** — PIX integrado, registro automático
- [ ] **App mobile** — PWA (sem custo de app store)

---

## 🔵 FASE 3 — Escala (Mês 4–6)

**Objetivo:** 100 clientes, R$10k MRR, sistema estável.

- [ ] **Multi-unidades** — dono de rede vê tudo num dashboard
- [ ] **Web3 Loyalty** — tokens na blockchain para fidelização
- [ ] **LAVAI Marketplace** — cliente procura lava-jatos próximos (como iFood)
- [ ] **API pública** — integrar com ERPs e sistemas legados
- [ ] **App nativo iOS/Android**
- [ ] **Painel de franquia** — lava-jatos que viram rede

---

## 🚀 FASE 4 — Plataforma Nacional (Mês 6–12)

**Objetivo:** Referência nacional, fundraising ou bootstrapping sustentável.

- [ ] **LAVAI Network** — rede de lava-jatos parceiros com padrão de qualidade
- [ ] **Analytics de mercado** — vender dados agregados para montadoras e seguradoras
- [ ] **Insurance integrado** — parceria com seguros auto
- [ ] **Expansão LATAM** — México, Argentina, Colômbia
